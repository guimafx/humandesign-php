# Checklist de preparação open source

## Concluído

- [x] repositório público
- [x] código versionado
- [x] testes locais
- [x] documentação básica

## Pendente

- [ ] decisão de licença
- [ ] revisão de compatibilidade com Swiss Ephemeris
- [ ] GitHub Actions confirmado
- [ ] múltiplos mapas de regressão
- [ ] API pública estável
- [ ] Composer
- [ ] release
- [ ] versionamento semântico
- [ ] política formal de manutenção

A estrutura da CI existe, compila uma versão fixada do Swiss Ephemeris e aguarda confirmação no primeiro push executado pelo GitHub Actions.
