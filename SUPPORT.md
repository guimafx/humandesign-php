# Suporte

- Bugs reproduzíveis: abra uma GitHub Issue com ambiente, driver e reprodução mínima.
- Dúvidas de uso: use GitHub Discussions, se estiver habilitado.
- Vulnerabilidades: siga [SECURITY.md](SECURITY.md) e prefira um Security Advisory privado.
- Pedidos de interpretação pessoal de mapa não pertencem ao core técnico e podem ser encerrados ou redirecionados.

O projeto e seus canais não oferecem aconselhamento médico, jurídico ou psicológico.
